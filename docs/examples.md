# Exemplos

O repositorio inclui exemplos para uso direto e para estudo da API.

## Script academico

O arquivo `examples/easy_usage.py` demonstra um notebook academico completo com:

- capa;
- secao;
- badges por nomes de cor;
- metricas;
- checklist;
- etapas;
- comparacao;
- tabela;
- figura;
- modulo com telas.

## Notebook

O arquivo `examples/notebookux_academic_example.ipynb` mostra a mesma proposta em formato de notebook.

## Showcase separado

O notebook `projeto_ia_diamonds_ufrpe (1).ipynb` pode ser mantido como referencia visual do projeto IA/Diamonds, fora do nucleo da biblioteca.
